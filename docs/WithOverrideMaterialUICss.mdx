# With OverrideMaterialUICss

This is the material-ui's button with some customised css created by [emotion](https://emotion.sh/):

```typescript
css(`
  color: red;
  background-color: blue;
  font-size: 4rem;
`)
```

We can change the button style.

<Playground>
  <OverrideMaterialUICss>
    <Button
      variant="contained"
      className={css(`
    color: red;
    background-color: blue;
    font-size: 4rem;
  `)}
    >
      Button
    </Button>
  </OverrideMaterialUICss>
</Playground>
